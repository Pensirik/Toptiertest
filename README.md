### install

- git clone https://github.com/Pensirik/Toptiertest.git
- cd to laravel project
- edit your .env file
- composer install
- php artisan key:generate
- php artisan serve

### mirgrate and seed user

- php artisan migrate
- php artisan db:seed --class=CreateUserSeeder
